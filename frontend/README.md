# AI Tools Frontend - Interface Futurista

Interface moderna e responsiva para o sistema AI Tools, com tema escuro e efeitos futuristas.

## 🎨 Características Visuais

### Tema Escuro Moderno
- **Fundo**: Gradiente escuro com animações sutis
- **Cores**: Paleta escura com acentos neon vibrantes
- **Contraste**: Alto contraste para melhor legibilidade
- **Glass Morphism**: Efeitos de vidro translúcido

### Efeitos Visuais
- **Partículas flutuantes**: Animações de partículas no fundo
- **Orbes luminosos**: Elementos decorativos com gradientes
- **Efeitos neon**: Bordas e sombras com brilho ciano
- **Animações suaves**: Transições fluidas em todos os elementos

### Responsividade
- **Mobile-first**: Design otimizado para dispositivos móveis
- **Tabs adaptativas**: Navegação por abas com scroll horizontal
- **Grid responsivo**: Layouts que se adaptam a diferentes telas
- **Touch-friendly**: Elementos otimizados para toque

## 🚀 Funcionalidades

### QR Code Generator
- Geração de QR codes com cache inteligente
- Download em PNG
- Impressão otimizada
- Interface intuitiva

### URL Shortener
- Encurtamento de URLs com analytics
- Copiar para clipboard
- Informações de cache
- Redirecionamento automático

### Emotion Analysis
- Detecção de emoções por IA
- Análise de múltiplas faces
- Resultados visuais interativos
- Overlay de análise em tempo real

### Demographics Analysis
- Análise de idade e gênero
- Detecção de características faciais
- Interface moderna para resultados
- Processamento otimizado

## 🛠️ Tecnologias

- **Tailwind CSS**: Estilização utilitária
- **CSS Custom Properties**: Variáveis para tema
- **CSS Animations**: Animações nativas
- **Responsive Design**: Design adaptativo

## 🔧 Configuração

### Instalação
```bash
npm install
```

### Desenvolvimento
```bash
npm start
```

### Build
```bash
npm run build
```

### Docker
```bash
docker build -t ai-tools-frontend .
docker run -p 3000:80 ai-tools-frontend
```
